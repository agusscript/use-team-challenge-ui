# Use Team challenge

Collaborative board in real time.

## Prerequisites

- [Node >= 18](https://github.com/nvm-sh/nvm) | This project was developed using node version `18.20.5`

## Project setup

```sh
npm install
```

## Environment variables

```sh
VITE_API_BASE_URL=
VITE_SERVER_SOCKET_URL=
```

## Compile and run the project

```sh
npm run start
```
